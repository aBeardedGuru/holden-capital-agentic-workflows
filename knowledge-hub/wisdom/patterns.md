# Wisdom Patterns

## Good Patterns

### Honest Deferral

The operator pauses to gather clarity rather than forcing conviction.

### Values-Constrained Choice

The operator accepts a less convenient path because it is more aligned.

### Clean Commitment

The operator only commits when capacity, integrity, and desire are aligned.

## Failure Patterns

### Hope-Based Decision

The operator mistakes optimism for evidence.

### Future Self Fiction

The operator assumes future capacity will be greater without reason.

### Rationalized Doubt

The operator notices real hesitation but reframes it as overthinking.
